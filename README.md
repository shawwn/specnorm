# specnorm

> Spectral norms

## Install

```
pip3 install -U specnorm
```

Or develop locally:

```
git clone https://github.com/shawwn/specnorm ~/specnorm
cd ~/specnorm
python3 setup.py develop
```

## Usage

```py
import specnorm

from specnorm import spectral_norm
import numpy as np

spectral_norm(np.ones((3,3,3)))
```
